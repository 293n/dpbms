DPBMS推定難易度表

発狂DP難易度表、 DP Overjoy、 DP Satellite を統合して計算された推定難易度表です。
難易度表だけでなく、プレイデータ読み込み、プレイヤーの能力推定、
それらに基づくフィルタリング機能、ビンゴ作成機能があります。

注意

  • 推定難易度は項目反応理論(IRT)を用いてLR2IRのデータから計算されています。
  • クリア系(Easy・Hard)とスコア系(AAA)の値がありますが、
    それぞれ独立した計算結果です。
  • 最小値が0、最大値が100になるようにスケーリングしています。
  • プレイ人数が少ない譜面では推定が不正確なことがあります。
  • DP Stellaには落ち着いたら対応します。

LR2 / beatoraja / LR2orajaのプレイデータ(.db)を読み込む

プレイデータを読み込み、プレイヤーレベルを推定することができます。

LR2:
/LR2files/Database/Score/[name].db
beatoraja / LR2oraja:
/player/player[number]/score.db
LR2IRから計算したデータなのでどちらかと言えばLR2、LR2oraja向けの機能です。
ゲージ仕様が異なるbeatorajaは参考程度ということでお願いします。
ここで読み込んだ結果は推定難易度表の値にフィードバックされません。
読み込まれたデータはブラウザのキャッシュに保存されます。
動作がおかしい場合はキャッシュをクリアしてみてください。
